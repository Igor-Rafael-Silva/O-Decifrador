# O Decifrador
A Máquina Enigma foi um dispositivo de cifragem eletromecânico utilizado pela Alemanha nazista durante a Segunda Guerra Mundial. Ela empregava rotores rotativos, um painel de conexões e um refletor para gerar bilhões de combinações possíveis, tornando as mensagens praticamente indecifráveis na época.

 Nós criamos um sistema parecido em códigos de programação em "C" e mostraremos um  pouco de como foi a organização e o desenvolvimento do sistema mostrando o Kanban, Documentação e Fluxograma.
